# Livro de Receitas Digital

Este é um projeto que fiz para treinar a base do React, e onde fiz o primeiro contacto com o Firebase. O meu foco foi aprender a ligar a aplicação a uma base de dados real e gerir a persistência de dados e imagens.

---

## Tecnologias utilizadas
- React.JS
- Firebase / FireStore
- Base64 Encoding

## O que aprendi
- **Integração com o FireStore:** Configurar o SDK do Firebase e fazer a gestão da coleção;
- **Imagens com Base64:** Aprender a lidar com o encoding de imagens para as guardar como texto na base de dados e renderizar novamente no ecrâ;
